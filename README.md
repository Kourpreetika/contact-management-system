## Key Features

1. Full CRUD operations for contacts
   - Add contact
   - View all contacts
   - Search contact
   - Edit contact
   - Delete contact

2. Menu-driven CLI interface

3. Persistent storage using `contacts.txt`

4. Duplicate phone number validation

5. Uses STL
   - `std::vector`
   - Iterators

6. File handling with streams
   - `ifstream`
   - `ofstream`

7. Proper input handling using
   - `cin`
   - `getline`
